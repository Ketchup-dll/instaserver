# InstaServer

A lightweight, dead-simple local HTTP server written in Go. Spin up a server for your static files instantly from the command line.
Go言語で書かれた軽量で超シンプルなローカルHTTPサーバー。コマンドラインから静的ファイル用のサーバーを即座に起動できます。

## Features

* **Zero Configuration:** Simply specify the folder, so no detailed configuration is required.
    設定要らず:ただフォルダを指定するだけなので細かな設定が必要ありません。
* **Single Binary:** No dependencies, no runtime required—just run the executable.
  　シングルバイナリ:依存関係なし、ランタイム不要、実行ファイルを直接実行するだけです。
* **Fast & Reliable:** Powered by Go's robust standard library.
  　高速で信頼性が高い:golangの堅牢な標準ライブラリで動作します。

# How to use

*Simply click instaserver.exe, then click the folder for the server you wish to set up. Ctrl-clicking the link displayed in cmd will take you to the local page.
*instaserver.exe をクリックし、設定したいサーバーのフォルダをクリックしてください。cmd に表示されるリンクを Ctrl キーを押しながらクリックすると、ローカルページに移動します。
